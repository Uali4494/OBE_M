PLO Level Reporting

  • PLOs Coverage: Report showing statistics of PLOs coverage intake wise
  • PLOs Skills Mapping: Report showing CLOs PLOs Skills Mapping courses wise
  • PLOs Analysis: Report showing all the PLOs statistics intake wise
  • PLOs Analysis Average Attainment: Report showing all the PLOs Analysis on Average Attainment of CLOs
  • PLOs Skills Coverage: Report showing statistics of skill coverage intake wise
  • PLOs Coverage Courses wise: Report showing statistics of PLOs coverage intake wise and courses wise

CLO Level Reporting

  • Offered Courses Marks Status: Report showing marks status of all offered courses for selected semester
  • Course Assessmnent Report (CAR): Report showing all course attainment
  • Compiled Result: Report showing course attainment
  • CLOs Mapping Courses wise: Report showing CLOs mapping over PLOs along with courses
  • CLOs Attainment Semester wise: Report showing CLOs Attainment along with courses
  • CLOs Failed: Report showing courses in which student cleared the course but failed CLOs for selected semester

CQI Level Reporting

  • CQI Course Executive Summary: Report Showing CQI cycle for the courses
  • CQI PLO Cohort Level Failure Summary: Report Showing CQI PLO Cohort Level Failure Summary for direct Assessment

Indirect Assessment Reporting

  • CSP Summary Report: Report Showing CSP Summary Intake wise
  • Exit Summary Report: Report Showing Exit Summary Intake wise
  • Internship Summary Report: Report Showing Internship Summary Intake wise
